《佛说自爱经》节选讲解
===============
经文：世尊又告王曰。众恶之罪最重有五。不孝不忠杀亲杀君。家灭国乱。重罪一也。
====
释义：此处说五种重罪，其第一条重罪，乃讲不孝父母、不忠于君王、杀害父母、杀害君王，导致家灭而国乱。这是重罪之一。好像家灭国乱，这四个字，适合给现在HK的有些人看看？


====
经文：罗汉之行得空不愿无想之定。与佛齐意拯济众生。而愚向之。重罪二也。
====
释义：这是说众生以愚蠢的行为对待阿罗汉，这话是佛对国王说的，也是对众生说的。
====


====
经文：佛者众罪已毕。景福会成。相好十力法导众生。慈悲喜护心过慈母。而愚恶谤。重罪三也。
====
释义：这一段是讲，众生以愚蠢的知见恶语诽谤于佛。
====


====
经文：清净沙门志清行高。怀抱经法助佛化愚。诸佛相绍继众生得度。皆由众僧。佞谗交搆以致不调。僧不调政法毁民狂走。政法毁民狂走者。三道兴恼比丘僧。重罪四也。
====
释义：这一条的核心，是讲“破和合僧”。清净沙门，以经法帮助佛度化凡愚。但是有人破和合僧，导致僧人不合，僧人不合用经法度化众生的事情无法顺利进行，于是则接下来就会发生政策毁坏民众利益的事情。而这都是因为兴起恼害比丘僧导致的。这是第四条重罪。
====



====
经文：佛之尊庙宝物水土。众生赤心以贡三尊。愚人或毁盗之。重罪五也。
====
释义：这一条的核心，是讲盗窃三宝财物，平时我们简单说，说盗僧祇物，完整的就是盗窃三宝财物。这是第五条重罪。
====



====
经文：犯斯五者。罪无请。谓之自杀身自灭族自投太山火矣。五罪之重重於须弥。慎无犯焉。佛说经竟。王及群臣皆得须陀洹。受五戒为清信士。国民有作沙门者。守戒为清信士者。遂以五戒十善为国政。诸天佑护国遂兴矣。诸天龙神王臣黎民。无不欢喜。
====
释义：犯了这五条重罪的人，忏悔都不能清净，所以叫做罪无请。犯这五条罪业的人，等于自己杀害自己，自己灭掉自己的家族，自己投生泰山地狱火中。五种罪业之重，超过须弥山的重量。慎重不要犯。佛说这部经结束时，国王和群臣都得到须陀洹果，然后受佛五戒。这里比较疑惑，难道证悟初果不需要持戒就可以证悟？不是别处经典，说一切大小乘果位都要持戒的基础吗？这里姑且存疑！我们暂时先按照这个意思来说。接下来，国民中做沙门的人，受戒的清净信士，都以五戒十善来建立国政。于是，诸天护佑，国家兴盛。
====




==============
（二）、关于提神的方法
==============

提问：师兄，有没有可以比较提神的方法呢？
====
回答：提升肾脏功能、提升培补精气神、除鬼神障、增益福德力、消除恶业力。除此之外，没有其他有意义的方法。比如吃货，看到美食就提神、比如颜控，看到美女就提神，总不是根本方法。以我个人经验，正常情况来说，我每晚只需要睡四个小时左右，就可以维系一天运转，就是按照上述方法做到的。


==============
（三）、复杂的修法誓约
==============

提问：《陀罗尼集经》佛顶心咒修法要求这么多，居然要求这么多，才能成就此咒法？而且要求不能去造经像家？难以理解。比如《陀罗尼集经》云：佛告诸比丘。此咒能解一切诸咒。若外道若摩醯首罗咒。亦能除却诸恶鬼神。亦救众生五苦八难。若善男子。至心受持佛顶心三昧陀罗尼咒。应当护持三业清净。三业清净有二种护。何等为二。一外护二内护。言外护者。不得食我世尊残食。不得食一切贤圣残食。不得食一切鬼神残食。不得食师僧父母残食。不得食一切众人残食。又不得食国王官人残食。不共众人传器而食。亦不得食毗那夜迦鬼魔之食。毗那夜迦食者。若面裹物。蒸煮烧熟。欢喜团等。皆不得食。若食此食。于三昧力不得成就。若一切人畜生产处不得往到。亦不得食。诸死亡家十恶家。酤酒家。五辛家。埋死人家。卖凶具家。不净人家。淫女家。造经像家。皆不得往。亦不得食。诸不净人。看他产人。捉死尸人。截割众生身肉之人。如是等人。皆不得　近与身相触。亦勿交往。此名外护清净之法。内护清净者。身不得杀生偷盗邪淫。口不妄语恶口两舌绮语戏论。皆不应作。意不应作贪嗔痴等。唯起大慈大悲大喜大舍等心。是名菩萨四无量心三业清净。由三业净。乃能受持此三昧陀罗尼佛顶咒印。此三昧陀罗尼力。悉能解除一切天魔外道咒法。皆能降伏一切怨敌。及摩醯首罗诸天鬼神所说咒术。悉能除灭。
========
========
吉祥观点：（1）、这就是我常说的持咒成就是三昧耶誓约成就、契约成就的深刻道理，所有的要求都是成就此三昧耶的具体契约条款，你能做到这些，修法持咒就会成就，做不到就不成就。这一段经文将所有的要求，详细列出，我反而觉得很好。（2）、至于造经像家，不准前往，这是修持此法的三昧耶之誓约的特定要求，大家只需遵守即可，不必责怪。（3）、这里是说专门以经营印造贩卖经像谋生的人家，类似今天造佛像工厂，以造像造经为产业谋生的，不可前往。这样的人家，确实有一些恶业。所以，我过去一直强调，经像建造流通，每个环节皆以成本建造流通即可，切切不可加入利益。《像法决疑经》曾经说过，买卖佛像者，五百世中常被人卖，也是这个意思。

（四）经典节选：
===========

《陀罗尼集经》若日日作种种供养阿弥陀佛。诵咒满十万遍。作印法者。即得灭罪命终生彼国。又若欲得生彼国者。亦更以泥作阿弥陀佛像十万躯。灭罪死生阿弥陀佛国。
