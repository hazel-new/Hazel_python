耍笑一番言之昭昭的“谛深大师”

耍笑一番言之昭昭的“谛深大师”
====================
谛深大师：谛深大师开示：
====
吉祥观点：敢问谛深大师：何为真谛？何见其深？何托其大？何德称师？

====
谛深大师：为什么依法不依人是邪说？
====
吉祥观点：因心邪故，必见“依法不依人”之正见为邪。如苏轼见佛印为牛粪，亦如无著见弥勒是病狗。依法不依人之说，出自《大般涅槃经》，谛深大师如此深入真谛，居然不知此为佛说，竟然斥为邪说。可见大师之心，其邪之甚。



====
谛深大师：坐下弟子，依法不依人是有漏说，无证说，罪孽说，世间法用说。入得佛门当知，依戒不依法，依法乃六道。
====
吉祥观点：（一）、既称众为坐下弟子，必是自视为“坐上之师”了？！（二）、敢问大师，既说依法不依人是罪孽说，有漏说。云何又说当依戒不依法？请教大师，戒是法乎？戒非法乎？戒若不是法，戒是何耶？问：戒若是法，则大师自立罪孽之说，戒若非法，则大师倡导非法，又是何耶？请大师不吝赐教一二？（三）、敢问大师，汝云“依法即是六道”之语，以何为证？黄毛小儿也知，依念佛之法，出六道生净土，云何依法乃六道？再问大师，依寂灭法，直趋佛道，六道又在何处？再问大师，依菩提心法，胜超一切小乘声闻功德，六道又在何方？大师名号倒是好混，然于教人，实乃误人。



====
谛深大师：佛于金刚经云，若有人说我有法可说，此人则为谤佛。实之例证，累世有之。
====
吉祥观点：佛说寂灭法时，大师以生灭法杂之，佛说生灭法时，大师以寂灭法杂之。大师若如此混淆视听，只怕连末学这一关也过不得，何敢妄称什么大师？敢问，若生灭寂灭二法不分，如大师意，引《金刚经》寂灭法义，说有法可说即为谤佛；而大师自己却自称要“依戒不依法”，此非有法可说乎？大师既有戒法可说，难道大师自己也是在谤佛吗？反之，若大师自诩无法可说，又亲说依戒之法，如此自打自嘴，不知为何？大师卖弄文字，看似雄辩滔滔，实则糊涂之极。

点评大师这段文字，我都感觉自贬身价。要不是有无聊佛友，发来这些粗鄙之语，实在不愿为此浪费时间。希望以后佛友，发点有水平的来博客，不要用这种垃圾开示污人眼目。



====
谛深大师：佛之末世，依依法不依人而堕落者不计其数。
====
吉祥观点：大师如此言之昭昭，说末法依法不依人而堕落者不计其数。敢请大师随意举例，有何人，依何佛法而不依人，而今堕落于何处？若大师能自证自语，我信汝诚实，若大师不能自证自语，我认汝为诈骗，何如？既然大师说不计其数，却列不出一个案例来证明，不知道所谓不计其数，从何说起？


====
谛深大师：因依法不依人立种种罪孽教派者，层出不穷，诸多邪魔外道皆因依法而立继而依人。
====
吉祥观点：（1）、同上问，既然大师信誓旦旦，说依法不依人立了什么罪孽教派，且层出不穷，不妨直接指名道姓，看看哪些教派是依法不依人的罪孽教派？不妨明说？都当了大师了，连这点丈夫胸襟都没有，指桑骂槐不敢明说，这也不是菩萨所为啊？既然为了利益众生，不妨说明你所说的邪魔教派何在？若说不明，是否可以认为大师“自靶自打”呢？（2）、法师反对“依法不依人”，可是又说“因依法而立继而依人”。若依法又依人，则已经违背依法不依人，于依法不依人有何关系呢？依法不依人，又为何要因之依人而蒙罪？


====
谛深大师：今者，如种种功派、种种所谓某法门而令大众受罪者，皆依法不依人之泛。
====
吉祥观点：首先佛教没有功派，不知大师说的是什么功派?大师是太极大师？还是气功大师？大师说某法门令大众受罪，而佛法没有令大众受罪之法门，不知大师所指为何？若是佛法，若非佛法，师当有明言，何必稀里糊涂！？还说什么依法不依人之泛滥？你连是不是佛法，都说不清，何谈依法？所依何法？又何谈依法不依人？又何谈依法不依人泛滥？
  

====     
谛深大师：佛涅磐时嘱，有佛住世以佛为师，佛灭度后以戒为师，是其者。
====
吉祥观点：大师倡导“以戒为师”，居然就要打倒“依法不依人”，且不知“戒即是法”，如此自相矛盾。如孝其母而杀其父，孝其父而杀其母，懵懂不知自身乃父母和合所生；如此蠢材，何堪法教？大师竟然不知“戒就是法”？何其荒唐！！敢问大师：若依法不依人是邪说，则以戒为师必是邪说，因戒也是法！！若以戒为师是正说，则依法不依人也是正说，因戒也是法！！大师连这点基本的逻辑都搞不清楚，基本的概念都不清楚，，还敢收什么“坐下弟子”？你这大师当得，真是醉生梦死！


====
谛深大师：是故，依戒不依人，为佛门说，依法不依人为外道典集、邪魔罪孽说。
====
吉祥观点：依戒为师，虽是佛说。然依法不依人，亦《大般涅槃经》中佛亲口所说，而绝非“谛深大师”所说的外道典籍、邪魔罪孽说。谛深如此说，等于直接咒骂佛为邪魔，咒骂《大般涅槃经》为外道典籍。如此愚痴之人，于佛教典籍一知半解，拾人牙慧学得三两言语，，就敢出来妄称大师？如此恶言咒骂释迦佛、诽谤《大般涅槃经》，不惧无间地狱乎？

痛劝大师，莫被虚名和钱财遮盖了眼目。对于你尚且还不懂的学问、不懂的知识、未证的境界，不明的佛理，不要出来乱讲一气。勿要谤佛谤法，误人误己！





========
老实结实（博客私信）：我还发现一个现象，对一个人有失偏颇的差评，也会带来非常严重的果报，恶疾等。一旦认识并消除，病会瞬间消失。举心动念，就这么厉害。
========
吉祥观点：

（1）、如是，如是。确实如此。评价一个人，或见其小有过失，即掩盖其一切功德，或见其小有功德，即掩盖其一切过失，二者俱偏颇。譬如网络上有批净空法师者，谓之邪师，此即偏颇之论。净空法师有不少见解严重错误，这是事实。但是净空法师带给佛教和大众的利益，依然是最主要的，这是我的评价。其余一切人等，无论出家在家，我的观点也是如此，功过分明，既不因功掩过，也不因过掩功。

（2）、佛教徒还有一个特殊性，见他人品行之过失，应当替人遮掩，这是没有问题的，隐恶扬善，理应如此。但是，如果对方知见上存在过失，理念上存在过失，是应该及时指出来的，这和隐人之恶的理念完全是两回事。譬喻：记得2012年4月我从普陀山经上海回陕西，我乘坐的大巴要开到上海南浦车站。当时车子到了长江北岸的慈城服务区，休息期间，有一辆私家车的车主要去普陀山，但是他居然是从宁波方向开过来的。他的车子应该从宁波直接开到舟山方向，但是他却开到上海方向来了，而且还过了杭州湾跨海大桥，到了上海这边。所以，我们大巴车的司机师傅，就告诉他，你方向走错了，应该原路返回宁波，然后往东开才能去舟山，然后才到普陀。告诉人开车方向走错了，提醒人走正确的方向，这就类似于及时指出佛教徒的知见问题，知见错误就是方向性的错误，一定要提出来，这样对方可以少走弯路，走错路。个人品德的过失，都可以替人隐藏，都可以隐人之恶扬人之善，但是涉及知见问题，一定要及时的指出，不然最后很麻烦。
